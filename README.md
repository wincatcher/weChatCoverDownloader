# weChatCoverDownloader
微信公众号文章封面下载

#一、缘起

每次转载公众号文章，下载原文封面总是个烦心事。

虽然已有现成方案如下：

1）右键查看源代码：

![](https://i.imgur.com/ijxpdPm.jpg)

2）Ctrl + F 搜索 “msg_cdn_url”：

![](https://i.imgur.com/tQPUf1u.jpg)

其后双引号中内容即为封面图网址，复制此网址后在浏览器中打开即可另存。

但毕竟过于繁琐，另外记忆搜索词也无意义，于是写了个一键下载工具分享给大家，已打包为 exe 文件，无需安装 Python 即可正常使用，后台回复 20180618 下载。


#二、方案
1、双击工具图标，弹出对话框

![](https://i.imgur.com/xti6LoO.jpg)

2、粘贴公众号文章链接并回车

![](https://i.imgur.com/GyNVEue.jpg)

3、文章封面自动下载至此文件夹，且以 “文章名称 - 封面” 模式命名。

![](https://i.imgur.com/YnXXaJz.jpg)
